About the Project

This project contains custom libraries/containers that I have worked on. Each of them are implemented based on the data structure and algorithms concepts.

Template data type

This is one of the most important and useful concepts I learn why building and implementing data structure and algorithms concepts. Template data type is like a parameter, where the data type will be determined as the functions or classes are called. This help programmer don't have to write multiple classes/algoritthms for different data types (int, char, objects, etc). It is a very useful tools and concepts to learn.

Iterator

Iterators are used for all the Libraries created. This helps with iterating through the container. Not only that, it provides an advantage in traversing through some containers such as Map, graph, etc. Iterators have common functions and feature that makes it useful, such as comparing operations, arithmetic operations, etc.
