Template data type

This is one of the most important and useful concepts I learn why building and implementing data structure and algorithms concepts. Template data type is like a parameter, where the data type will be determined as the functions or classes are called. This help programmer don't have to write multiple classes/algoritthms for different data types (int, char, objects, etc). It is a very useful tools and concepts to learn.
